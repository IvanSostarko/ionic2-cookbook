#Setup

## Install Cordova
```sh
npm install -g cordova
```

## Install Ionic 2 
```sh
npm install -g ionic@beta
```

## Clone repository
```sh
git clone https://github.com/IvanSostarko/ionic2-cookbook.git
```

## Start project
```sh
ionic serve
```
